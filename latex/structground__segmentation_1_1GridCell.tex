\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename PointT$>$\newline
struct ground\+\_\+segmentation\+::\+Grid\+Cell$<$ Point\+T $>$}
Represents one discretized 3D voxel in the spatial grid. 

Each cell stores\+:
\begin{DoxyItemize}
\item Raw points assigned to the voxel
\item PCA eigenvalues/eigenvectors
\item Estimated surface normal
\item Plane fitting inliers
\item Terrain classification
\end{DoxyItemize}

Cells are classified using geometric consistency, slope, and region growing.
